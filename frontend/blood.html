Service 01 · Random Forest

Blood Report Analyser

Translate the nine standard Complete Blood Count values from a routine pathology report into a probabilistic estimate of likely haematological category, with an explanatory note and aligned dietary guidance.

Input

Enter your CBC values.

Reference adult values are pre-filled — overwrite with the figures from your own report.

About This Tool

Random Forest classifier ensembling 200 decision trees, benchmarked against logistic regression, gradient boosting and support vector machines.

  • Five target classes: Normal, Anemia, Infection, Leukopenia, Thrombocytopenia.
  • Returns full probability distribution alongside top prediction.
  • Feature importances align with clinical intuition.
Reference Ranges

Haemoglobin 13.5-17.5 (M) · 12.0-15.5 (F). RBC 4.5-5.9 (M) · 4.0-5.2 (F). WBC 4-11. Platelets 150-450. Values outside these ranges may indicate underlying conditions worth discussing with a clinician.

Methodology

Trained on data/blood_report_dataset.xlsx with cleaning, median imputation, repeated model trials, and tuning documented in 01_blood_report_analyzer.ipynb.

Important

A normal CBC does not exclude all illness, and an abnormal CBC does not always indicate disease. This tool is informational only — confirm findings with a qualified clinician.
