Service 03 · CNN with Transfer Learning

Image Diagnosis

Submit a chest radiograph or a dermatological photograph. A Convolutional Neural Network fine-tuned from a MobileNetV2 backbone returns the predicted class and a full softmax distribution.

Input

Upload an image.

Select the model type appropriate for your image, then choose a file from your device.

Architecture

MobileNetV2 backbone pretrained on ImageNet, followed by global average pooling, dropout regularisation, and a softmax classification head. Two-phase training: frozen backbone, then partial unfreezing for fine-tuning.

  • X-ray: binary classification trained on chest radiographs.
  • Skin: five-class lesion identification.
  • Inputs resized and normalised; augmentation at training only.
Image Quality Matters

Models trained on clean datasets degrade rapidly on out-of-distribution inputs. Use clear, well-lit, properly oriented images for best results.

Transfer Learning

Starting from ImageNet-pretrained weights enables strong performance even with modest medical datasets — the lower layers detect edges and textures common to all imagery.

Not Diagnostic

A CNN classification is a statistical guess, not a diagnosis. Any clinically concerning image must be reviewed by a qualified radiologist or dermatologist.
